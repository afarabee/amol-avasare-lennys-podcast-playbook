# Deliberate Money Left on the Table

## Classification
Doctrine, not a skill.

## Why it moved out of /skills
This is a decision principle that should shape growth, experimentation, and monetization behavior broadly. It is not a standalone triggered procedure.

## Core doctrine
Not all profitable moves are worth making. Short-term metric gains can damage long-term trust, safety, brand, user experience, or mission.

## Use this doctrine when deciding
- whether to run a controversial growth test
- whether to prioritize trust over short-term revenue
- whether a move violates a red line or merely requires stronger justification

## Operational lens
Classify opportunities as:
- red line / do not run
- high sensitivity / only test with strong rationale
- acceptable

## Implication
A mature growth system is willing to forgo local gains in order to preserve durable strategic assets.
