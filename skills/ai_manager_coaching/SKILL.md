# AI Manager Coaching

## Purpose
Use this skill to prepare for coaching, feedback, and management reflection using structured AI synthesis.

## Use this skill when
- A manager wants weekly coaching support
- There is abundant communication and project context
- A leader wants help spotting patterns across direct reports or themselves
- A user asks for a structured AI-enabled management cadence

## Do not use this skill when
- There is no meaningful performance or context data
- The output would be used as the sole basis for performance judgment
- Sensitive feedback requires direct human investigation first

## Core principles
1. Use AI to synthesize, not to judge definitively.
2. Ask for patterns, observations, and possible coaching angles.
3. Ground outputs in goals, work artifacts, and recent discussions.
4. Treat AI as a smart but imperfect coach.
5. Always apply human judgment before acting.

## Workflow
1. Gather inputs:
   - team goals
   - OKRs
   - work summaries
   - discussion transcripts
   - manager priorities
2. Ask for:
   - key takeaways
   - potential blind spots
   - feedback themes
   - suggested coaching questions
3. Separate strong signals from speculative suggestions.
4. Use outputs to prepare 1:1s or self-review.
5. Refine prompts over time based on false positives and false negatives.

## Output format
Return:
- observed patterns
- possible concerns
- growth opportunities
- coaching prompts
- confidence notes
- suggested follow-ups

## Anti-patterns
- Treating AI feedback as fact
- Delivering raw AI language directly to reports
- Using weak context and expecting strong guidance
- Ignoring privacy, sensitivity, and context boundaries
